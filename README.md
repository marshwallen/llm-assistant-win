## Windows 集成 DeepSeek 智能分析插件
### 功能规划
- **前端**：
1. 以桌面悬浮窗口的形式常驻桌面，按x键可最小化（如windows右下角的显示隐藏图标区域）。
2. 功能包含类似对话的交互式页面。

- **后端**：
1. 日志监控：事件查看器分析+内存转储（Memory Dump）分析
2. 硬件监控：通过CPU利用率和频率，GPU利用率和频率，系统温度、内存利用率和硬盘利用率趋势等分析硬件使用情况。因为要知道趋势，需要使用某些数据库保存近1天每秒的上述数据。数据库表：短期（保存一天，间隔1秒，86400条），中期（保存7天，间隔10秒，60480条），长期（保存1个月，间隔1分钟，43200条）
3. 文件结构分析：用户可能需要分析硬盘的文件结构和大小，或者知道哪些文件占地方，哪些文件较大等
4. 调度策略：遇到突发性能需求时，自动调整windows电源计划（最佳能效、平衡、最佳性能）。更智能化，不是根据哪个进程来的。（数据驱动）

### 注意事项
注意：用户可在任意一台windows笔记本安装此软件，通过 HTTP/gRPC 或其他方式高效访问处于另一台设备上的 Ollama 服务以使用 DeepSeek。这里需要使用某些技术来减少延时。
